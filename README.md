## About

Frontend app for Pokemon NFT-marketplace.

Stack:

* NextJS
* GraphQL
* Apollo

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn dev

# production mode
$ yarn build
```
